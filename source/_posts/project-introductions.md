## Vocab Buddy

### 项目介绍

该项目是一款基于Web的在线生词默写系统，为广大学生群体提供便利的英语单词默写及听写服务。

### 技术路线

#### 前端

- TypeScript
- Vue 3 with Composition API
- VueRouter
- Pinia with Composition API
- Naive UI

#### 后端

- Java 17
- Spring Boot 3
- MyBatis Plus

### 项目情况

目前完成情况为听写核心功能，后续需要补充基础功能，并优化听写测试的生成策略。
